1.Copy the Contents inside a folder.
2.Make sure you have Node and npm installed and if not installed, install it first. Use npm install in the CMD in the project directory where the package file exists.
This installs the required packages for the project.
3.Create a OpenWeather account from Openweathermap.org and do the modifications to the code by using ur credentials in "app.js" file. The guidelines are commented for ur convenience and displayed there.
3.Host it in Heroku, Netlify or Firebase (Much more if you are familiar with them).


Note: While hosting on Heroku, you need to create a Procfile in the home project directory. Similar things have to followed when hosting on different sites. This project is a Nodejs Live Weather Display project.

Screenshots have been added in the home directory after the successful testing in local host.
